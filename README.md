# Bank Loan Performance Analysis
Bank Loan Performance Analysis using SQL and Power BI

## Project Overview
This project analyzes Bank Loan data to understand Loan Performance, Repayment Behavior, and Potential Risk Factors.
The dataset contains 38,567 loan records. SQL was used to explore and analyze the data, and Power BI was used to build an interactive dashboard to monitor loan performance.

## Tools Used
- SQL – data querying and analysis
- Power BI – dashboard creation and data visualization

## Dataset
The dataset contains 38,567 Loan Records including information about Loan Amount, Interest Rate, Borrower Income, and Loan Status.

## Key KPIs
- Total Loan Applications: 38.6K
- Total Funded Amount: $435.8M
- Total Received Amount: $473.1M
- Average Interest Rate: 12.0%
- Average Debt-to-Income (DTI): 13.3%

## Key Insights
- The majority of loans fall under the "Fully Paid" category, indicating strong repayment performance.
- A smaller portion of loans are classified as "Charged Off", representing potential credit risk.
- Loan applications increased over time, showing growth in lending activity.
- Borrowers with higher debt-to-income ratios tend to have higher default risk.

## Dashboard
This Power BI dashboard summarizes loan applications, funded amounts, repayment performance, and risk indicators.
